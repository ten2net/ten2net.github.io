Opencv在Ubuntu上的安装过程

21 Dec, 2016

一、阅读 二、安装依赖 To install OpenCV 2.4.2 or 2.4.3 on the Ubuntu 12.04 operating system, first install a developer environment to build OpenCV. apt-get -y install build-essential cmake pkg-config Install Image I/O libraries apt-get -y install libjpeg62-dev apt-get -y install libtiff4-dev libjasper-dev Install the GTK dev library apt-get -y install libgtk2.0-dev Install Video I/O libraries apt-get -y install libavcodec-dev libavformat-dev libswscale-dev libv4l-dev Optional - install support for Firewire video cameras apt-get -y install libdc1394-22-dev Optional - install video streaming libraries apt-get -y install libxine-dev libgstreamer0.
